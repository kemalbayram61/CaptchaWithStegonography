# CaptchaWithStegonography
 Combining the stegonography method and the captcha method.

Captcha işlemi ile Stegonografi işleminin birleştirilmesi.

Bu proje resim captcha işlemi yapılmaktadır. Seçilecek olan resimlere aynı zamanda şifreli olarak kullanıcı bilgileride gizlenmektedir 
bu sayede hem kullanıcının insan olduğu hemde o kişi olduğu kanıtlanmaktadır.
